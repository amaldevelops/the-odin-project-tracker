Odin Project Foundations Course: Revisiting Rock Paper Scissors, V2.0

Odin Project instructions

Version 1.0 of this game could only be played from the console. This version will allow the game to play from the Browser UI

Instructions to play the game

  • Player will have five rounds to play against the computer and in the end it will be shown who won the competition
  • Click on the below three buttons to select Rock, Paper or Scissors
  • Computer will randomly generate a choice and will be compared against the user choice
  • Final result will be shown on the screen
